About Alvanklaveren.com

Alvanklaveren.com is created, written and maintained by A.L. van Klaveren. It is (currently) built in Java (EE), using Apache Hibernate, Apache Wicket, and Spring Framework, based on a three-tier and service-oriented architecture. The website is running on Wildfly 10.1 on a public hosting platform called OpenShift v3. Last but not least, the website's layout is supported by Bootstrap.
